# Campo-Minado-em-CPP

Este é um jogo de **Campo Minado** clássico desenvolvido para o terminal como projeto académico da faculdade. O objetivo é demonstrar lógica de programação, manipulação de matrizes e geração de números aleatórios em C++.

## Funcionalidades
- Tabuleiro 6x6 (ajustável no código).
- Geração aleatória de minas a cada partida.
- Contador de bombas adjacentes.
- Interface simples via console.

## Tecnologias Utilizadas
- Linguagem: **C++**
- Bibliotecas: `iostream`, `vector`, `ctime`, `cstdlib`.
